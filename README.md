# xinghongma.github.io

Personal portfolio site for Xinghong Ma, built with React, TypeScript, Vite, and Tailwind CSS.

## Local development

Prerequisite: Node.js 20+

1. Install dependencies with `npm ci`
2. Start the dev server with `npm run dev`

## Deployment

This repository is configured for GitHub Pages via GitHub Actions.

For the main user site URL, publish this code from the repository named `xinghongma.github.io`.
